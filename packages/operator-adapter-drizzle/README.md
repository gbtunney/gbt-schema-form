# @operator/adapter-drizzle

> Drizzle ORM persistence adapter implementing OperatorStore

This package provides a persistence adapter using the Drizzle ORM. It implements the interfaces defined in
`@operator/store` and depends on the core types in `@operator/core`. Use this adapter in projects that persist
operator data to PostgreSQL or SQLite via Drizzle.
